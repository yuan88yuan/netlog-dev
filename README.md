# Netlog Dev

This project is a web application built with the following technologies:

- **Backend:** Cloudflare Worker
- **Database:** Cloudflare D1
- **Frontend:** React (with Vite) + TypeScript
- **UI Framework:** Bootstrap 5

## Development

### Frontend

To start the frontend development server, run the following commands:

```bash
cd frontend
npm install
npm run dev
```

### Backend

To start the backend development server, run the following commands:

```bash
cd backend
npm install
npx wrangler dev
```
